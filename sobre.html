Sobre este site

Olá. Criei este espaço para partilhar produtos que realmente uso ou recomendo: livros que li, tech para o escritório em casa, ideias para organizar e presentes que funcionam. Tudo com opinião honesta — não recomendo nada só para vender.

Os links para a Amazon são de afiliado. Isso significa que, se comprares através deles, recebo uma pequena comissão sem custo adicional para ti. Ajuda a manter o site e a continuar a testar e a escrever sobre produtos. Obrigado pelo apoio.

Encontrarás listas temáticas (melhores livros, essenciais para casa, tech, presentes até 20€) e reviews do que comprei recentemente. Se tiveres sugestões de temas ou produtos que queiras ver aqui, podes usar o link do programa de associados da Amazon no rodapé para saber mais sobre como funciona.
